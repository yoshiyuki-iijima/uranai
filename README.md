# 東洋占術アプリ

算命学・四柱推命・易学を統合した占いWebアプリ。生年月日と性別から鑑定結果を生成します。

## デプロイ手順（ローカル環境不要）

PCにNode.jsやGitをインストールせず、ブラウザだけでGitHub + Vercelに公開できます。詳細手順はチャットの会話を参照してください。要約：

1. GitHubでリポジトリを新規作成
2. このプロジェクトのファイルをWeb UIでアップロード
3. Vercelでリポジトリをインポートして自動デプロイ

## 技術スタック

- Next.js 14 (App Router) / TypeScript
- フロントエンド・APIともに同一プロジェクト内
- Vercelに最適化（追加設定不要）

## ロジック構成

| モジュール | 役割 |
|---|---|
| `lib/kanshi.ts` | 干支計算の共通基盤（ユリウス通日ベースの日干支、立春境界の年干支、五虎遁の月干支） |
| `lib/shichusuimei.ts` | 四柱推命：三柱（年月日）、通変星、十二運、五行バランス、大運方向 |
| `lib/sanmeigaku.ts` | 算命学：陰占（干支）と陽占（北方・中央・南方の三星） |
| `lib/ekigaku.ts` | 易学：梅花心易による64卦・動爻の算出 |
| `lib/fortune.ts` | 3システムを統合したリーディング |

## 入力仕様

- 生年月日（1900-01-01 ～ 2100-12-31）
- 性別（male / female）

## API仕様

### POST /api/fortune

リクエスト：
```json
{ "birthYear": 1990, "birthMonth": 5, "birthDay": 15, "gender": "male" }
```

レスポンス：四柱推命・算命学・易学の全結果と統合リーディングを含むJSON。

## 拡張の方向性

- 出生時刻対応（時柱の追加）
- 大運・年運の表示
- 節入日の太陽黄経による精密計算
- 鑑定文のLLM生成
